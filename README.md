# VASD

Flutter приложение / дипломный проект для колледжа.

## Как запустить на компьютере

Этот проект готов к запуску почти сразу после клонирования на компьютер:
- ```console
  git clone https://github.com/GASTDASH/vasd.git
  ```
- ```console
  cd vasd
  ```
- ```console
  flutter pub get
  ```
- ```console
  flutter run
  ```
